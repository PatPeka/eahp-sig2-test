## Viewing the interactive agenda online

The agenda in `01-meetings/call-01/` can be published with GitHub Pages so it can be opened in a browser from a public web URL.

To enable GitHub Pages:

1. Open this repository on GitHub.
2. Go to **Settings**.
3. In the left sidebar, open **Pages**.
4. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
5. Choose the `main` branch and the `/ (root)` folder, then save.
6. After GitHub finishes publishing, open the Pages URL and browse to `01-meetings/call-01/`.

The agenda page will be available at a URL like:

`https://patpeka.github.io/eahp-sig2-test/01-meetings/call-01/`
